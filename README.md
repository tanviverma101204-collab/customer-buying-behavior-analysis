# customer-buying-behavior-analysis
SQL | Python | Power BI Customer Buying Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI. The objective is to identify customer purchasing patterns, sales trends, product performance, and business insights that can help improve decision-making.

## Tools & Technologies
- Python (Pandas)
- MySQL
- Power BI
- VS Code
- GitHub

## Dataset
Customer Shopping Behavior Dataset

## Project Workflow
- Data Cleaning using Python
- Exploratory Data Analysis (EDA)
- SQL Business Queries
- Interactive Power BI Dashboard

## Key Insights
- Revenue comparison by gender
- Top-selling products
- Customer segmentation
- Subscriber vs Non-subscriber spending
- Seasonal sales analysis
- Shipping preference analysis
- Payment method analysis
- Discount usage analysis
